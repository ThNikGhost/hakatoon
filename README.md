## Getting Started

Follow the following instructions



### Installation




1. Clone the repo
   ```sh
   git clone https://github.com/ThNikGhost/hakatoon.git
   ```
2. Install requirements
   ```sh
   pip install -r requirements.txt
   ```
3. Start the server 
   ```sh
   python manage.py runserver
   ```
